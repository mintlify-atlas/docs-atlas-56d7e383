## Welcome to AWS Certified App

AWS Certified App is an interactive web application designed to help you prepare for the **AWS Certified Cloud Practitioner** certification exam. Built with React 19, Vite, and Tailwind CSS, this comprehensive practice exam simulator provides realistic exam conditions with questions organized by the official AWS exam domains.

## Practice modes

The application offers two distinct practice modes to suit your preparation needs:

### Domain mastery tests

Focus on specific domains with targeted 30-question quizzes that provide immediate feedback. Each domain test covers:

- **Domain 1: Cloud Concepts** (24% of exam) - 30 questions
- **Domain 2: Security and Compliance** (30% of exam) - 30 questions
- **Domain 3: Cloud Technology and Services** (34% of exam) - 30 questions
- **Domain 4: Billing, Pricing, and Support** (12% of exam) - 30 questions

<Note>
  Domain tests show explanations immediately after answering each question, making them ideal for learning while practicing.
</Note>

### Full exam simulation

Experience realistic exam conditions with:

- 65 randomized questions from all domains
- 90 minutes to complete (matches actual exam time)
- Official domain distribution: 16, 20, 21, and 8 questions per domain
- 15 unscored questions included (simulating AWS's experimental questions)
- No feedback until completion
- Review screen before submitting

<Warning>
  In full exam mode, you cannot see answers or explanations until you complete and submit the entire test.
</Warning>

## Study features

The application includes powerful features to enhance your study experience:

### Question controls

- **Flag questions**: Mark questions for later review with a single click
- **Add notes**: Create personal notes for any question with a built-in editor
- **Eliminate answers**: Exclude wrong answers to narrow down choices
- **Navigation grid**: Jump directly to any question using the sidebar

### Question indicators

Questions are color-coded in the navigation grid:

- 🟡 Yellow/Orange: Flagged for review
- 🟢 Green: Correct answer
- 🔴 Red: Incorrect answer
- ⚪ White: Unanswered
- 🔵 Blue dot: Has personal notes

### Security features

To simulate real exam conditions:

- Screenshot prevention during active tests
- Copy/paste blocking while taking exams
- Right-click disabled during tests
- Focus on genuine learning without shortcuts

<Info>
  These security features are automatically disabled when you finish the exam or return to the menu.
</Info>

## Performance analytics

Track your progress with comprehensive performance metrics:

- **Circular progress indicators**: Visual representation of your score per domain
- **Domain breakdown**: See exactly which areas need improvement
- **Last exam summary**: Review your most recent full exam performance
- **Historical tracking**: Your performance data persists across sessions

<Tip>
  Domain cards show green progress circles for scores ≥80% and red for scores below 80%, helping you quickly identify weak areas.
</Tip>

## Exam information

The AWS Certified Cloud Practitioner exam consists of:

- **65 questions total**: 50 scored + 15 unscored experimental questions
- **90 minutes**: Test duration (approximately 1.38 minutes per question)
- **Passing score**: 700 out of 1000 (roughly 70%)
- **Question types**: Multiple choice (one correct) and multiple response (two or more correct)

<Warning>
  AWS recommends consistently scoring at least 80% in practice tests before attempting the official certification exam.
</Warning>
